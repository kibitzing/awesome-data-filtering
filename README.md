# awesome-llm-data
A repository of information about data used in training large language models (LLMs)


## Evaluation dataset
### **Bias**:
*  [Bold](https://arxiv.org/pdf/2101.11718)
  * Used by Llama 2
 
### **Truthfulness**: 
* [TruthfulQA](https://arxiv.org/pdf/2109.07958)
  * Used by Llama 2
 
### **Toxicity**:
* [ToxiGen](https://arxiv.org/pdf/2203.09509)
  * Used by Llama 2
